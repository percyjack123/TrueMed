TrueMed is an AI-powered web application for detecting counterfeit medicine
packaging using deep learning and OCR-based batch verification.

## Features
- CNN-based visual authenticity detection
- OCR-based batch ID extraction
- CSV-based batch verification
- Explainable AI results
- React + Flask web interface

## Model Hosting
The trained model is hosted on Hugging Face and loaded dynamically at runtime.

## Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Flask, TensorFlow
- OCR: Tesseract
- Model Hosting: Hugging Face
